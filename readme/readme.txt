使用说明

测试接口请用Rest Client进行测试，请求头部的验证已关闭

本地测试请按开发文档说明搭建本地环境

为了方便也可在公网直接进行访问和测试API的URL地址：
http://yii.ljzxzxl.com/rest/index.php/

可参考下面的测试用例：
测试请求						请求方式
http://yii.ljzxzxl.com/rest/index.php/user/list		GET
http://yii.ljzxzxl.com/rest/index.php/user/create	POST
http://yii.ljzxzxl.com/rest/index.php/user/view/1	GET
http://yii.ljzxzxl.com/rest/index.php/user/update/1	PUT
http://yii.ljzxzxl.com/rest/index.php/user/delete/1	DELETE

其他详细请参考文档说明！
